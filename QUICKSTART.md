# FamMap Quick Start Guide

Welcome to FamMap! This guide will help you get started with discovering kid-friendly locations.

## 📱 Accessing FamMap

### Web Browser
Visit: [FamMap Website](https://fammap.vercel.app) (when deployed)

### Mobile App
FamMap is a Progressive Web App (PWA):
1. Open in your mobile browser
2. Tap "Add to Home Screen" or "Install App"
3. FamMap will be available offline

## 🗺️ Getting Started

### 1. Discover Locations

**On the Map:**
- Zoom in/out to explore locations
- Tap any marker to see location details
- Use "Find Me" to see nearby locations

**Search:**
- Type location name or category
- Filter by type (Parks, Restaurants, Medical, etc.)
- Toggle "Stroller Friendly" for accessible routes

### 2. View Location Details

When you tap a location marker:
- See full details with photos
- Check available facilities
- Read user reviews and ratings
- View operating hours and contact info
- See dining options and parking info
- Check nursing room availability

### 3. Save Your Favorites

**Add to Favorites:**
- Tap the heart icon (❤️) on any location
- Access your saved places anytime

**View Favorites:**
- Tap "Favorites" button in sidebar
- See all your saved locations

## ⭐ Reviews & Ratings

### Leave a Review

1. Select a location
2. Tap "Write Review"
3. Rate (1-5 stars)
4. Add comments about:
   - Cleanliness
   - Facilities quality
   - Staff friendliness
   - Child-friendliness
5. Add photos (optional)
6. Submit

### Read Reviews

- See other parents' experiences
- Read specific feedback about facilities
- Check recent ratings
- Look for facility-specific comments

## 📊 Report Crowdedness

Help other families plan better:

1. Visit a location
2. Tap "Report Crowdedness"
3. Rate current crowd level (1-5)
4. Add optional comment
5. Submit

This helps other families plan visits during less crowded times.

## 🔔 Get Alerts

### Set Up Alerts

1. Go to "Alert Center"
2. Select locations you want updates for
3. Choose alert types:
   - Operating hours changes
   - New reviews/ratings
   - Facility updates
   - Events happening
   - Price changes
4. Save preferences

### Manage Alerts

- Tap "Alerts" to see all notifications
- Remove alerts you no longer need
- Adjust notification frequency

## 🌐 Language Support

### Change Language

## 🔍 Search Tips

### Find Locations Faster

**By Name:**
- Type location name exactly
- Search works while you type
- See suggestions appear

**By Category:**
- "Parks" - Playgrounds, gardens
- "Restaurants" - Dining with kids
- "Nursing Rooms" - Baby facilities
- "Medical" - Hospitals, clinics
- "Attractions" - Entertainment

**By Facilities:**
- Check "Stroller Friendly"
- Filter by "High Chair Available"
- Find "Changing Tables"
- Look for "Nursing Rooms"

### Filter Results

1. Select category
2. Enable facility filters
3. See matching locations on map
4. Sort by: Distance, Rating, or Name

## 🆘 Help & Support

### Common Questions

**Q: Can I use FamMap offline?**
A: Yes! After your first visit, the app works offline with cached location data.

**Q: How do I add a new location?**
A: Tap the "+" button in the main view or use "Add Location" in the menu.

**Q: Can I edit someone else's review?**
A: No, only review the location yourself. Venue managers can respond to reviews.

**Q: How accurate is the crowdedness data?**
A: Based on real-time reports from users visiting the location.

**Q: Can I share my favorite locations?**
A: Yes! Each location has a share button for social media or messaging.

### Report Issues

Found a problem? Help us improve:

1. Tap menu ≡
2. Select "Report Issue"
3. Describe the problem
4. Include location/feature if applicable
5. Submit

Our team will investigate and follow up.
