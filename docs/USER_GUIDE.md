### 本地端（家裡）

- **後端 API**: http://localhost:3001

### 外部訪問（需要 Cloudflare Tunnel）
- 啟動後會提供一個 https://xxx.trycloudflare.com 網址

## 功能說明

### 1️⃣ 切換城市
- 預設顯示 **台北市** 景點
- 支援切換至 **新北市**、**桃園市**
- 點擊左上角「城市選擇器」即可切換

### 2️⃣ 景點類型過濾
- **餐廳** 🍜
- **景點** 🏛️
- **親子** 👶
- **公園** 🌳
- 點擊分類按鈕篩選想要查看的類型

### 3️⃣ 互動地圖
- **點擊Marker**: 查看景點詳情（名稱、地址、評分）
- **拖曳**: 移動地圖
- **縮放**: 放大/縮小地圖範圍

### 4️⃣ 搜尋功能
- 點擊搜尋框，輸入關鍵字（景點名稱或地址）

## 常見問題

### Q: 地圖都沒顯示景點？
A: 請確認：
1. 後端服務是否運作（Port 3001）
2. 嘗試點擊不同分類切換
3. 移動地圖範圍（景點可能載入中）

### Q: Marker 點了沒反應？
A: 可能是 UI 重疊問題，嘗試：
1. 重新整理頁面 (F5)
2. 切換分類後再點擊

### Q: 如何回報問題？
A: 請截圖並描述操作步驟，發給小龍蝦 🦞
